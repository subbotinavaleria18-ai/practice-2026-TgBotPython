## 1. Общая информация о проекте

**Название проекта:** «Steamleaf — симулятор раздачи листовок»

**Цель проекта:** Разработать к 25 мая 2026 года полностью играбельный прототип инди-симулятора «Симулятор раздачи листовок», оснащённый базовым геймплеем и пользовательским интерфейсом.

**Задачи проекта:**
1. Провести анализ предметной области и изучить игровые механики, характерные для жанра симуляторов.
2. Сформировать концепцию проекта, разработать сценарий и описать правила взаимодействия игрока с игровой средой.
3. Спроектировать архитектуру проекта и программно реализовать игровые механики с использованием игрового движка.
4. Создать визуальный контент (2D/3D графика) и разработать пользовательский интерфейс.
5. Провести тестирование разработанного прототипа и подготовить отчётную документацию по итогам работы.
