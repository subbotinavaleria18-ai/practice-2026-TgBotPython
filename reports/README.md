## 3. Техническое руководство по Telegram-боту

| Файл | Описание |
|------|----------|
| `telegram-bot-guide.md` | **Техническое руководство по созданию Telegram-бота «Steamleaf»** — содержит исследование предметной области, архитектуру, пошаговые инструкции, примеры кода, UML-диаграммы, скриншоты, описание модификации проекта и заключение.

**Содержание руководства:**
- Исследование предметной области и анализ существующих решений
- Архитектура технологии (UML-диаграммы, жизненный цикл команд)
- Пошаговое руководство для начинающих (регистрация бота, установка, написание кода)
- Создание бота «Steamleaf» с 5 командами
- Модификация проекта (визуальное сопровождение, улучшение галереи)
- Заключение
